# decodelabs_tasks


## 📝 Todo Task List App

A modern desktop To-Do List application built with Python and CustomTkinter, featuring a clean claymorphism-inspired user interface and task management functionality.

## 🚀 Features

- ✅ Add new tasks
- 🗑 Delete tasks
- ✔ Mark tasks as complete
- ⚠ Automatic overdue task detection
- 🎯 Task priority management (High / Medium / Low)
- 🔍 Filter tasks by status
- 💾 Persistent local storage using JSON
- 🎨 Modern claymorphism-inspired UI
- 📜 Scrollable responsive task board

---

## 🛠 Tech Stack

- **Python**
- **CustomTkinter**
- **Tkinter**
- **JSON**

## ⚙ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/decodelabs_tasks.git
cd decodelabs_tasks
```

### 2. Install dependencies

```bash
pip install customtkinter
```

### 3. Run the application

```bash
python todo_app.py
```
